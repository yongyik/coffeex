# MyCoffee - Coffee Shop Website 

一个使用 Next.js 制作的咖啡店展示网站，适合作为咖啡馆、饮品店、小型餐饮店的官网案例。-

## 功能

- 首页 Hero 展示
- 推荐饮品 Swiper 轮播
- 菜单分类与价格展示
- 饮品详情弹窗
- 关于我们页面
- 联系页面
- WhatsApp 询问表单
- Google Map 嵌入
- 隐私政策页面
- SEO metadata
- sitemap.xml / robots.txt

## 技术

- Next.js
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- Swiper

## 页面

- `/`
- `/about`
- `/menu`
- `/contact`
- `/privacy`

## 说明

此项目为作品集案例，用于展示咖啡店官网的设计与开发能力。
